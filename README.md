# Baky Search

Vite + React + Tailwind demo app (BAKY Search). Ready for deployment to Vercel.

Quick start:

```bash
cd "C:\Users\Martin\Downloads\baky-search"
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Vercel: connect this repository and set the build command to `npm run build` and the output directory to `dist`.
